## How TALON Works

```
10+ Job Sources → LLM Scoring (Gemini/NVIDIA/DeepSeek)
       ↓
  Filter & Rank → Resume Tailoring (GPT-4o)
       ↓
  Gmail Monitor → CRM Tracking → Discord Alerts
       ↓
  News Intel → Daily Report → GitHub Commit (← you are here)
```
